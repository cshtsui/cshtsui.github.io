Usage

List is a simple layout component used to display a sequence of items. It is used along with ListItem (as children) to render unordered list (<ul>) elements without any of the styling that would normally come with it (e.g. bullets or large left-padding).

With Dividers

The List component can optionally include dividers between items which will add a light border to the top of each list item except the first. This is useful for more complex list items that may need a further separation affordance than simple spacing.

The List component should not be used to create a standard typographical bulleted list. Instead, just use a <ul> element with <li> elements within. Forge includes standard styles for these elements as well as the <ol> (ordered list) element.

Design

The List component can be used to display a collection of items that have no numerical ordering and would not make sense in a bulleted list format. For example, it could be used to display a list of sub-navigation items or a collection of providers with profile images.

Implementation

The List component creates a <ul> element. ListItem components should be included as children and will wrap their children in a <li> element.

To add a divider between ListItem components, set the dividers prop on the List component to true.

API Tables

Import List and ListItem Only

Importing individual components can reduce bundle size. Learn more

import List from '@athena/forge/lib/list';
import ListItem from '@athena/forge/lib/list-item';

List Props

Prop Type Default Description
className string Adds a class to the root element of the component
dividers bool false
padded bool true Adds default padding that can be removed when the parent already provides a gutter
...rest Passthrough props to the <ul> element

ListItem Props

Prop Type Default Description
className string Adds a class to the root element of the component
...rest Passthrough props to the <li> element

HTML API

Class Applies to Description
fe_c_list <ul> element Base class required for list
fe_c_list--dividers <ul> element Adds a border-top to all fe_c_list__item elements except the first
fe_c_list--padded <ul> element Adds padding to the list
fe_c_list__item <li> element Base class required for list item
