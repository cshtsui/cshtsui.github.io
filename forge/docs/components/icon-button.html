Usage

An IconButton might be used in the same cases as Button, but where additional visual emphasis needs to be provided by an icon. Follow the usage guidance for Button.

Design

For guidance on how and when to use icons, follow the design guidance for the use of Icon. In general, making an icon into an IconButton should be limited to cases where extra visual emphasis is necessary and not merely decorative.

Button text should use title case (e.g., "Add Diagnosis", not "Add diagnosis").

If you are using IconButton as part of a form, you can find more guidance on the Form Design Foundations page.

Alerting Icons and IconButton

Do not use the special information icons within IconButton (i.e., Attention, Info, Success, Critical). These icons are meant to emphasize key information and should not be combined into a single, call-to-action element.

Implementation

In some cases (e.g., using React Router's Link component), it may be necessary to style a link directly with an icon. To do this, use IconButton classes with a link element via className and pass in the Icon directly to the link:

<a className="fe_c_icon-button">
  <Icon icon="Add" />
  <span className="fe_c_icon-button__text">Add</span>
</a>

Import IconButton Only

Importing individual components can reduce bundle size. Learn more

import IconButton from '@athena/forge/lib/icon-button';

IconButton Props

Prop Type Default Description
icon (required) string The icon to display inside the button. Note it is not recommended to use the solid, special icons (Attention, Critical, Info, and Success) within IconButton
iconSet objectOf
  • IconBase components
 ForgeIcons Specify a custom icon set defined as a dictionary of IconBase components. Icon sets can be generated using the App Fabric Forge Extension Generator.
aria-label string Description of the button's action for accessibility; defaults to icon's title if the button has no text
className string Adds a class to the root element of the component
disabled bool false Applies the disabled state class and adds disabled attribute to the button element
onClick func Handles click event on the button
text string Button text
title string Sets the title prop on the <button> element and the inline content of the<title> element in the svg; defaults to the icon's name
type string button Sets the type attribute on the button element (e.g., button, submit, or reset)
...rest Passthrough props to <button>
