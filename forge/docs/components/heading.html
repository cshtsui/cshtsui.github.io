Heading

Headings introduce sections of content and establish a visual and semantic structure for page content.

Demos

  • HTML

      <Heading headingTag="h3" variant="subsection" text="Subsection Heading" />

}

Usage

Headings in Forge separate the visual appearance of the font from the semantic heading hierarchy. Traditional CSS frameworks merge the two concepts by defining styles for the <h1>…<h6> HTML elements. Doing so may lead to a conflict between needing a certain semantic hierarchy (e.g., for accessibility or search engine optimization) and visual treatment.

Page Headings

There should only be one page heading per page with a recommended character count of no more than 24 characters (including spaces and punctuation).

Section and Subsection Headings

Use as many section and subsection headings as needed.

Heading Descriptions

Use heading descriptions to add brief supplementary descriptions related to a heading underneath it. Heading descriptions are for content that would otherwise make the primary heading too lengthy for quick scanning of the page. However, do not use heading descriptions to write lengthy paragraphs.

Design

Since Forge decouples visuals from semantic concepts, it’s up to the content designer or developer to determine an appropriate HTML heading element in headingTag for each heading. In most cases, page headings are h1 or h2, section headings are h3 and higher, and subsection headings are h4 and higher. Assistive technologies such as screen readers use these hierarchies to determine a logical read order for their users, so proper ordering is important.

Headings should use title case (e.g., "Patient Profile Details", not "Patient profile details").

Heading props

Prop Type Default Description
className string Adds a class to the root element of the component
headingDescription string Optional description text that goes below the heading text
headingTag enum
  • 'h1'
  • 'h2'
  • 'h3'
  • 'h4'
  • 'h5'
  • 'h6'
 'h1' Specifies the heading's HTML tag. For more on the importance of heading tags in semantic HTML, see the W3C's article on this topic.
text string The heading text
variant enum
  • 'page'
  • 'section'
  • 'subsection'
 'page' The visual style of the heading
...rest Passthrough props to the heading HTML element
