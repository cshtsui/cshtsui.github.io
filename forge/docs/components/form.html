Usage

ReduxForm provides greater built-in functionality than Form, including features like validation and state management via integration with the Redux application architecture. If you do not have strong implementation reasons for using Form, ReduxForm is recommended instead.

Form is meant to be used with FormField.

Design

Forms are commonly assembled using a variety of input components (e.g. text input, textarea, select, checkbox, etc). Forge provides a recommended responsive layout for these inputs via the FormField and ReduxFormField components. Additionally, FormField bundles a given input with a label, a component for error messages, and a component for informational (hint) text. FormField also provides styling for common cases such as indicating a field is required. Be sure to review these provided conveniences before deciding to construct a form manually.

Find more guidance on the Form Design Foundations page.

Implementation

The Form component provides a wrapper around the native html <form> element. It includes a submit button by default with customizable text. Users should use Form as a container for FormField components if they do not use Redux to manage state.

Customizing the Button Row

To use a custom Button row instead of the default button:

  1. Set the includeSubmitButton prop to false to turn off the default.
  2. Pass the custom button row component into the inputSlot prop of the FormFieldLayout component. Using FormFieldLayout will keep the button row in alignment with the other FormFields.
    <FormFieldLayout inputSlot={<MyButtonRow />} />

Form Props

Prop Type Default Description
buttonText string 'Submit' Text used on the submit button
className string Adds a class to the root element of the component
compact bool Reduces bottom margin of form fields.
includeSubmitButton bool true Whether the default submit button should be used
labelAlwaysAbove bool Makes labels always go above inputs. Overrides labelWidth.
labelWidth number Width of the field labels in grid columns. Defaults to 3/12.
nested bool If true, removes standard grid row outer margins. Set to false if the form is at the top level of page layout. Defaults to true.
noValidate bool true Prevents browser validation because it triggers unwanted error popups and interferes with ReduxForm validation.
onSubmit func Function called when the user clicks a submit button in a form. Passed a submit event parameter.
...rest Passthrough props to the <form> element

HTML API

The form element doesn't have any classes. It is just an empty container.
