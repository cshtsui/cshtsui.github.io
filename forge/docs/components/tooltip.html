Usage

Tooltips provide additional help for an item. They may offer a definition or give a hint of the functionality associated with the element being hovered over.

Do not use the Tooltip component to display critical information. Depending on its placement option, Tooltip may appear clipped or outside of the viewport (e.g., a top placement for Tooltip on an input that is already at the top of the viewport would open outside the visible area).

Design

Tooltips contain read-only text and should be as short as possible. Text should use sentence case.

The Tooltip component is strictly for information display and does not support having interactive elements within it.

For a placement of top and bottom, the tip is horizontally centered against the width of the widest child of Tooltip. For a placement of left and right, the tip is vertically centered against the total height of all children of Tooltip.

Implementation

Tooltip can have any number of children. Hovering over any of the children in this element will trigger Tooltip. All Tooltip children are currently wrapped in an inline-block element for the purpose of determining Tooltip placement. Be aware that this can cause some unintended positioning issues when used within display: flex elements.

To control when Tooltip is triggered instead of relying on hover, set showTip. Never set showTip={true} on initial render because the placement will be wrong if layout hasn't completed yet.

Tooltip Props

Prop Type Default Description
text (required) string The text displayed in the tooltip
children node The target component, element, or text that will have the tooltip applied to it
className string Adds a class to the root element of the component
placement enum
  • 'top'
  • 'bottom'
  • 'left'
  • 'right'
 'top' The placement of the tooltip around its target
showTip bool Controls whether the tooltip is visible. When undefined tooltip is visible on hover. Do not set to true on initial render.
theme custom: ThemeConsumer.themePropType Theme variables for configuring an instance of Forge. See Root component for more details.
...rest Passthrough props to React Overlay, a third party component used to position the tooltip
